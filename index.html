Project

GETOPTIMUM

Optimum is a decentralized network protocol for web3 that speeds up gossip data propagation across nodes worldwide using Random Linear Network Coding (RLNC) — a coding technique developed under the supervision of Professor Muriel Médard (MIT). Its flagship product is mump2p: a fast, fault-tolerant protocol for delivering blocks, blobs, and transactions, compatible with libp2p and GossipSub.

Pac-Man Optimum is a fun side project from the brand — a nod to the ∞ptimum name, not directly tied to the protocol itself.
