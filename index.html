GETOPTIMUM

Optimum — децентрализованный сетевой протокол для web3, ускоряющий передачу данных (gossip) между узлами по всему миру с помощью Random Linear Network Coding (RLNC) — технологии кодирования, которую развивали под руководством профессора Мюриэль Медар (MIT). Флагманский продукт — mump2p: быстрый и отказоустойчивый протокол доставки блоков, блобов и транзакций, совместимый с libp2p и GossipSub.

Pac-Man Optimum — игровой сайд-проект бренда: дань фирменному ∞ptimum, не связанная напрямую с самим протоколом.
